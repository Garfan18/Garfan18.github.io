Jump'n'run

First Unity project

Project Information

  • Project Type: Mario Party like Jump'n'run
  • Date: December 2020
  • Used Programs: Unity, Blender
  • Role: Solo project

Overview

I started taking programming classes to learn the basics in order to have a more adequate start towards my game design degree. With that in mind we looked at Unity and worked on this project. In order not to develop a new game from scratch but rather work with something pre-existing and well known we used Super Mario as a template and tried to recode it. This allowed me to implement typical game mechanics such as collecting coins, losing lives and travelling through pipes in a Unity project using scripts and animations. I also worked with Blender for the project and used it to 3D model the enemy „Gumba“ for the game.
