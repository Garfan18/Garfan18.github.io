The art of Film

Turning an article into a website

Project Information

  • Project Type: Website
  • Date: April 2023
  • Used Programs: Visual Studio CodeS
  • Role: Solo project

Overview

In the KickWeb module, we acquired the skills to code using HTML, CSS and JavaScript for the first time. To avoid undue stress, we were given the task of choosing a magazine to convert into a website. The initial task was to simply program a website. I opted for the magazine 'The Art of Film' due to the impressive visual elements it featured. This initial foray into web programming proved to be a source of pride for me, as I found the process of designing and developing my website to be highly satisfying. The highlight of the website is the Millennium Falcon mouse pointer, which allows users to fly through the site. A key challenge of the project was to ensure responsive programming for tablets and mobile devices.
