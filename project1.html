Meow or Never

2D Pixel Art Puzzle Game

Project Information

  • Project Type: Puzzle Game
  • Date: March 2025
  • Used Programs: Unity, Photoshop, Aseprite
  • My Role: Game Designer, Pixel Artist

Project Description

"Meow or Never" is a 2D pixel art puzzle game where players guide a mischievous cat through increasingly complex puzzle environments. I focused on designing intuitive yet challenging puzzles, creating charming pixel art, and ensuring smooth player controls.
